**# Atliq-Hardware-SQL-Business-Insights**

## Project Overview
This project is part of the Codebasics SQL Resume Challenge in the Consumer Goods domain.

The objective of this project is to analyze AtliQ Hardware business data using SQL and generate meaningful business insights to support data-driven decision-making.

The analysis focuses on:
- Sales Performance
- Product Analysis
- Customer Insights
- Channel Contribution
- Market Performance
- Growth Trends

---

## Tools Used
- MySQL
- Excel
- PowerPoint

---

## Dataset Overview
The dataset contains:
- Fact and Dimension tables
- Sales data for FY2020 & FY2021
- 27 Markets
- 4 Regions
- 397 Products
- 209 Customers

---

## Key Insights
- FY2021 gross sales showed strong YoY growth.
- Retailer channel contributed more than 73% of total sales.
- AQ Gamers Ms became the top-selling product.
- Accessories segment recorded the highest product growth.
- Q1 achieved the highest sold quantity in FY2020.
